## About GRaTeR-JAX

GRaTeR-JAX is a JAX-ed version of the GRaTer scattered light (debris) disk modeling code, originally from Augereau et al. 1999 and translated to Python by Julien Milli in the vip-hci package's disk forward modeling module (https://vip.readthedocs.io/en/latest/tutorials/05B_fm_disks.html). Authors: Mihir Kondapalli, Max Millar-Blanchaer, Briley Lewis
